x-spreadsheet is a A web-based JavaScript spreadsheet.
This overview details all stand out features in x-spreadsheet
from Grouping, Pivoting, Pagination, Tree Data and many more.

#### Features Overview

* Undo & Redo
* Paint format
* Clear format
* Format
* Font
* Font size
* Font bold
* Font italic
* Underline
* Strike
* Text color
* Fill color
* Borders
* Merge cells
* Align
* Text wrapping
* Freeze cell
* Functions
* Resize row-height, col-width
* Copy, Cut, Paste
* Autofill
* Insert row, column
* Delete row, column
* hide row, column
* multiple sheets
* print
* Data validations
